PR

PRul se ocupă de imaginea firmei, de relația dintre aceasta și publicul ei, de felul în care ajung mesajele la public și analizează felul în care sunt percepute acestea. Pe scurt, PRul are grijă de reputația companiei.
Departamentul de PR este cel care lucrează împreună cu celelalte departamente astfel încât imaginea companiei să fie una pozitivă și relația acesteia cu publicul să fie una cât mai înfloritoare.
